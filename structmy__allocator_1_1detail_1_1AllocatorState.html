Detailed Description

Internal shared state of the allocator.

Stores logical allocation accounting shared between all allocator copies referring to the same arena.

  • max_elements_ – maximum number of elements allowed (0 means unlimited)
  • allocated_ – number of elements logically allocated

Copies of the allocator share this state via std::shared_ptr, meaning allocation limits are shared across containers using the same allocator instance.
